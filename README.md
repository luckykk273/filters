# filters
Several classical filters for IMU(or MARG) sensors.


## Required sensors
| Filters              | Accelerometer | Gyroscope | Magnetometer | 
| :------------------: | :-----------: | :-------: | :----------: |

| Complementary Filter | &#9711;       | &#9711;   | &#9651;      |
| Madgwick             | &#9711;       | &#9711;   | &#9651;      |
